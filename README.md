# BDB3


## Build
1. Install [Node](https://nodejs.org/), [Bower](http://bower.io/), [Sass](http://sass-lang.com/), and [Gulp](http://gulpjs.com/).

2. Install Node and Bower packages.
```sh
npm install
bower install
```

3. Run the Gulp build or Gulp watch task.
```sh
gulp
```
```sh
gulp watch
```
